#Simple Blog.

A minimal blog theme for Drupal 7 Blog. It can also be used as a great starter theme if you're familiar with Drupal 7's theming engine.

Some included theme building tools are SASS support, Autoprefixing, Compass SASS mixins, BrowserSync and gulp.
